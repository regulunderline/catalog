Запуск:  
  cd .\frontend  
  npm install  
  npm run build  
  npm run preview  
    
Запуск в dev mode:  
  cd .\frontend  
  npm install  
  npm run dev  

Тестирование:  
  cd .\frontend  
  npm test  

.env file:  
  VITE_API_URL="https://fakestoreapi.com/products"  


Использованные технологии:  
  Redux, Redux Toolkit: упрощение State Management и масштабирования приложения, предотвращение Prop Drilling.  
  TypeScript: Strict Mode для повышенной надежности и читаемости кода, выявления ошибок на этапе написания.  
  tailwindcss: ускорение верстки и упрощение работы с css прямо внутри HTML-кода.  
  Vite: упрощение сборки и настройки приложения.  
  Axios: упрощение запросов к api.  
  use-debounce: Маленькая библиотека (< 1 Kb), позволяющая легко осуществлять dedounce.
